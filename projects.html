HRA SMS Pilot – Overview

This pilot uses a realistic clinical scoring algorithm to prioritize outreach to high-risk, low-engagement Dual-SNP members.

  • Problem: Low HRA completion rates among vulnerable members.
  • Goal: Improve STARS rating via predictive segmentation and automated outreach.
  • Result: ~30% more high-risk members targeted than with random outreach.

Technical Breakdown

This simulation was developed using Python (Pandas, Plotly) to:

  • Clean and normalize mock DSNP member data.
  • Score risk using factors like chronic conditions, age, recency.
  • Stratify members into outreach tiers.
  • Visualize insight via interactive charts.
# Scoring logic in sms_scoring.py
  df['score'] = (
    (1 / (df['last_hra_days'] + 1)) * 50 +
    (df['response_history'] * 10) +
    (df['chronic_conditions'] * 30) +
    (df['age'] > 75).astype(int) * 15 +
    (df['dual_eligible'] * 5) +
    (df['snp_member'] * 10)
  )

This ensures outreach focuses on members overdue for an HRA and at highest clinical/socioeconomic risk.
