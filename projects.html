HRA SMS Pilot – Overview

This pilot uses a realistic clinical scoring algorithm to prioritize outreach to high-risk, low-engagement Dual-SNP members.

  • Problem: Low HRA completion rates among vulnerable members.
  • Goal: Improve STARS rating via predictive segmentation and automated outreach.
  • Result: ~30% more high-risk members targeted than with random outreach.

Technical Breakdown: Member Scoring Logic

The following Python script was used to generate risk scores and SMS eligibility for over 2,000 DSNP members. The logic prioritizes overdue, high-risk, low-engagement individuals for HRA outreach

1. Load Cleaned Member Data

Started by importing Pandas and reading the cleaned dataset from clean_sms_data.py.

import pandas as pd
            df = pd.read_csv('data/sms_members_cleaned.csv')

2. Score Each Member

This formual assigns a score based on time since the members last HRA, chronic conditions, past engagement, and socioeconomic risk factors. All values are additive.

df['score'] = (
          (1 / (df['last_hra_days'] + 1)) * 50 +         # More recent HRA → lower score
          (df['response_history'] * 10) +                # Past engagement ↑ score
          (df['chronic_conditions'] * 30) +              # Each condition adds weight
          (df['age'] > 75).astype(int) * 15 +            # Elderly flag bonus
          (df['dual_eligible'] * 5) +                    # Socioeconomic risk
          (df['snp_member'] * 10)                        # Plan targeting boost
      )

3. Statify by Risk

This splits the population into three tiers using quantile bins (equal-sized groups):

df['risk_tier'] = pd.qcut(df['score'], q=3, labels=['Low', 'Medium', 'High'])

4. Flag Members for SMS Outreach

This logic identifies high-priority members based on a blend of risk, plan statuse, and timing.

df['eligible_for_sms'] = (
        (df['snp_member'] == 1) &
        (df['dual_eligible'] == 1) &
        (df['last_hra_days'] > 100) &
        (df['response_history'] <= 2) &
        (df['age'] >= 65) &
        (df['chronic_conditions'] >= 2)
    )

5. Rank Members and Save Output

The members are sorted by score (descending) and saved as a ranked file for visualization and outreach planning.

df = df.sort_values(by='score', ascending=False)
      df.to_csv('data/sms_outreach_ranked.csv', index=False)

This scoring engine powers the interactive plots shown in the Overview tab, ensuring every visual is grounded in real targeting logic.
